Google Photos Remote Media Entity

This module helps you to have Googole Photos as Media source in Drupal Media.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/media_entity_remote_google_photos

 * To submit bug reports and feature suggestions, or track changes:
   https://www.drupal.org/project/issues/media_entity_remote_google_photos
